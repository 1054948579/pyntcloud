=============================
Making point clouds fun again
=============================

.. image:: /docs/images/pyntcloud_logo.png

Pyntcloud is a Python 3 library for working with 3D point clouds.


Requirements
============
* Python3
* numpy
* scipy
* pandas
* ipython
* pytest
* matplotlib
* numba
* python-lzf
* laspy

Setup
============
```
pip install git+git://github.com/daavoo/pyntcloud.git
```
